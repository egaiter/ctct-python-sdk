Result_Set

class Result_Set(dict)
Result_Set is a subclass of the basic python dictionary type and can use any method available to standard python dictionaries.
Result_Set.get_set()
Returns a list of results.
Result_Set.get_item(index)
Returns a single result. index accepts an integer.
Result_Set.set_type()
Returns the type of the Result_Set's objects.
Result_Set.add_item(item)
item accepts anything that is the same type as the result set type.
Result_Set.remove_item(index)
index accepts an integer.
Result_Set.clear_result_set()
Removes all results and sets the set type to None.
Result_Set.get_next_page()
Gets the next page URI if it exists.
