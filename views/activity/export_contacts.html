Activity Endpoints

ConstantContact.export_contacts(columns, lists, sort_by = 'EMAIL_ADDRESS', file_type = 'CSV', export_added_by = False, export_date_added = False)
Returns a shortened
Activity_Status
object with relevant information to the request. columns accepts a list of strings specifying which properties to include in the exported file. lists accepts a list of strings representing valid contact_list ids or system list names specifying which lists to export. Currently, the API only supports one list id. sort_by accepts a string defining the sort order of the exported file. file_type accepts a string specifying the format of the export file. Currently, only CSV is supported. export_added_by accepts a boolean when set to true adds a column showing who added or removed the contact to or from the list. export_date_added accepts a boolean when set to true adds a column showing the date that each contact was added.
