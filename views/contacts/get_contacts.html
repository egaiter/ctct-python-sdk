Contacts Endpoints

ConstantContact.get_contacts(email = None, limit = 50, modified_since = None, status = 'ALL')
Returns a
Result_Set
of
Contact
objects. email accepts an
Email_Address
or a string and passes it as a URL query specidying a specific contact to retrieve. limit accepts an integer and passes it as a URL query specifying the maximum number of contacts per page. modified_since accepts an iso-8601 formatted time string and passes it as a URL query to retrieve contacts created after a specific date. status and passes it as a URL query to retrieve contacts with a specific status.
