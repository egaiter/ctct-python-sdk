class Attribute(dict)
Attribute is a subclass of the basic python dictionary type and can use any method available to standard python dictionaries. Any method that uses an Attribute model can also accept a properly formatted standard python dictionary. The following attributes have get, set, and delete methods (get_{attrName}, set_{attrName}, delete_{attrName}):
