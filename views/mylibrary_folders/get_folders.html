MyLibrary Folder Endpoints

ConstantContact.get_folders(limit = 50, sort_by = 'CREATED_DATE_DESC')
Returns a
Result_Set
of
Mylibrary_Folder
objects. limit accepts an integer and is passed as a URL parameter specifying the maximum number of folders to include per page. sort_by accepts a string and is passed as a URL parameter specifying how to sorty the folders.
