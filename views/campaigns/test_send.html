Campaigns Endpoints

ConstantContact.test_send_campaign(campaign_id, email_addresses, format, personal_message)
Returns an object containing the initial test send request body. campaign_id accepts a
Campaign
object or string. email_addresses accepts a list of either
Email_Address
objects or strings and specities the recipients of the test send. format accepts a string and specifies whether to send the HTML content, TEXT content, or both in the test email. personal_message accepts a string to be included with the test email.
