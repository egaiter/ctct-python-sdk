Campaigns Endpoints

ConstantContact.get_campaigns(limit = 50, modified_since = None, status = 'ALL')
Returns a
Result_Set
of
Campaign
objects. limit accepts a non-zero positive integer and is passed as a URL parameter. modified_since accepts an iso-8601 formatted date as a string and is passed as a URL parameter. status accepts a string and is passed as a URL parameter.
