Getting Started

Welcome to the Constant Contact Python SDK help pages!
Using the SDK is super easy. If you haven't already, you can get started using the Constant Contact API by following the directions on our developer portal. The libraries can be obtained through pip
$ pip install constantcontact
And here's a short code sample to get you started
> from constantcontact import ConstantContact > ctct = ConstantContact({api_key}, {access_token}) > print json.dumps(ctct.get_account_info(), indent=4) { "website": "constantcontact.com", "organization_name": "ConstantContact", "time_zone": "US/Eastern", "first_name": "Lambert", "last_name": "Wang", "email": "lambert_wang@example.com", "phone": "1234567890", "company_logo": "", "country_code": "US", "state_code": "MA", "organization_addresses": [ { "city": "Waltham", "line1": "1601 Trapelo rd", "postal_code": "02451", "country_code": "US", "state_code": "MA", "state": "Massachusetts" } ] }
Good luck and have fun!
