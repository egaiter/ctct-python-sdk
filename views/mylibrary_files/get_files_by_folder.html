MyLibrary File Endpoints

ConstantContact.get_folder_files(folder_id, limit = 50, sort_by = 'CREATED_DATE_DESC', file_type = 'ALL')
Returns a
Result_Set
of
Mylibrary_Folder
objects. folder_id accepts a
Mylibrary_Folder
object or a string. limit accepts an integer and is passed as a URL parameter specifying the maximum number of files to include per page. sort_by accepts a string and is passed as a URL parameter specifying how to sort the results. file_type accepts a string and is passed as a URL parameter to retrieve files of a specific type.
