MyLibrary File Endpoints

ConstantContact.get_files(limit = 50, sort_by = 'CREATED_DATE_DESC', source = 'ALL', file_type = 'ALL')
Returns a
Result_Set
of
Mylibrary_File
objects. limit accepts an integer and is passed as a URL parameter specifying the maximum number of files to include per page. sort_by accepts a string and is passed as a URL parameter defining how to sort the results. source accepts a string and is passed as a URL parameter to retrieve files uploaded from a specific source. file_type accepts a string and is passed as a URL parameter to retrieve a specific type of file.
